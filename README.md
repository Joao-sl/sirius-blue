1. Download the ZIP file from [GitHub](https://github.com/Joao-sl/sirius-blue/archive/refs/heads/main.zip) by clicking on the Code button and Download ZIP
2. Extract the `theme` folder
3. Search for `chrome://extensions/` in chrome address bar
4. Turn on the `Developer mode` on the top right of the screen
5. Click on `Load unpacked` and select the `theme` file
6. Enjoy! (You can turn off the Developer mode if you want.)
